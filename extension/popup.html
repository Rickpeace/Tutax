Steply Recorder

Nehmen Sie einen kurzen Bildschirm-Screencast auf. Ihre Klicks werden mitgezaehlt und als exakte Schrittgrenzen an Steply uebergeben.

Direkt-Upload zu Steply (optional)

Fuegen Sie den Verbindungs-Token aus Steply (Einstellungen → Einbetten → Steply Recorder verbinden) ein. Dann landet die Aufnahme automatisch in Steply - ohne Datei-Umweg. Ohne Token erhalten Sie zwei Dateien zum Hochladen.

So geht's:
  1. Oeffnen Sie den Tab, den Sie erklaeren moechten.
  2. Klicken Sie auf „Aufnahme starten" - ein Aufnahme-Tab oeffnet sich.
  3. Waehlen Sie im Dialog den Tab bzw. das Fenster zum Aufnehmen.
  4. Fuehren Sie Ihre Schritte vor, dann „Aufnahme beenden".
  5. Laden Sie beide Dateien in Steply hoch (Aus Video).

Klicks werden im aufgenommenen Browser-Tab erfasst - auch ueber Seitenwechsel hinweg. Wechseln Sie in einen anderen Tab oder ein anderes Programm, fehlen dort die Klicks; das Video wird aber vollstaendig aufgenommen.
