Steply Recorder

Nehmen Sie einen kurzen Bildschirm-Screencast auf. Ihre Klicks werden mitgezaehlt und als exakte Schrittgrenzen an Steply uebergeben.

So geht's:
  1. Oeffnen Sie den Tab, den Sie erklaeren moechten.
  2. Klicken Sie auf „Aufnahme starten" - ein Aufnahme-Tab oeffnet sich.
  3. Waehlen Sie im Dialog den Tab bzw. das Fenster zum Aufnehmen.
  4. Fuehren Sie Ihre Schritte vor, dann „Aufnahme beenden".
  5. Laden Sie beide Dateien in Steply hoch (Aus Video).

v1: Klicks werden nur im gestarteten Browser-Tab erfasst. Wechseln Sie den Tab oder das Programm, fehlen dort die Klicks - das Video wird aber vollstaendig aufgenommen.
